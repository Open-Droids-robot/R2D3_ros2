# r2d3_mujoco Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** A self-contained `r2d3_mujoco` package giving the R2D3 robot a full-parity MuJoCo simulation backend (controllers + lidar + IMU + RGB-D + Nav2/SLAM + MoveIt), mirroring the existing Gazebo package `dual_rm_simulation`.

**Architecture:** Pure resource package (no compiled code) built on the installed `mujoco_ros2_control` 0.0.3 (ros-controls). URDF→MJCF conversion happens at launch via the upstream converter, wrapped by a caching script (`ensure_mjcf.py`) that hashes the xacro output and skips conversion when nothing changed. Sensors publish on the same topics as the Gazebo sim, so Nav2/SLAM/MoveIt configs are reused unmodified from `dual_rm_navigation` / `r2d3_bringup`.

**Tech Stack:** ROS 2 Jazzy (primary) + Humble, `mujoco_ros2_control` 0.0.3, xacro, `imu_sensor_broadcaster`, `depth_image_proc`, Python 3 (stdlib `unittest` — pytest is not installed on this machine).

**Approved spec:** `docs/superpowers/specs/2026-07-11-r2d3-mujoco-design.md`

## Global Constraints

- Do **not** modify any existing package (`dual_rm_simulation`, `dual_rm_navigation`, `r2d3_bringup`, `dual_rm_description`, moveit configs). Depend on them only.
- Package location: `r2d3_mujoco/` at repo top level (`/home/samzpc/code/r2d3/src/R2D3_ros2/r2d3_mujoco/`).
- Workspace root: `~/code/r2d3` (already built; `install/` exists). Every runtime command needs `source /opt/ros/jazzy/setup.bash && source ~/code/r2d3/install/setup.bash` first.
- Must work on Humble too: guard distro differences with `os.environ.get("ROS_DISTRO") == "humble"` (upstream demo pattern); no Jazzy-only launch API; tests use stdlib `unittest`, not pytest.
- Topic parity (exact names): `/clock`, `/scan`, `/imu`, `/camera/image`, `/camera/depth_image`, `/camera/camera_info`, `/camera/points`, `/joint_states`, `/odom` (+TF from `diff_drive_controller`).
- MJCF cache location: `$ROS_HOME/r2d3_mujoco/<robot_model>/` (`$ROS_HOME` defaults to `~/.ros`). Converter's final output file inside it: `mujoco_description_formatted.xml`.
- MJCF topic: `/mujoco_robot_description`, QoS RELIABLE + TRANSIENT_LOCAL, depth 1 (must match `mujoco_ros2_control`'s subscriber).
- 0.0.3 sensor naming conventions (verified from installed headers/demos):
  - IMU: MJCF sensors `imu_sensor_quat`/`imu_sensor_gyro`/`imu_sensor_accel`; ros2_control `<sensor name="imu_sensor_imu">`.
  - Camera: ros2_control `<sensor name="camera">` — name must equal the MJCF camera name; params `frame_name`, `info_topic`, `image_topic`, `depth_topic`; hardware param `camera_publish_rate`.
  - Lidar: ros2_control `<sensor name="lidar">` — matches the raw `<rangefinder name="lidar" site="rf"/>`; params `frame_name`, `angle_increment`, `min_angle`, `max_angle`, `range_min`, `range_max`, `laserscan_topic`; hardware param `lidar_publish_rate`.
  - Free joint added by the converter is named `floating_base_joint` (URDF side gets `virtual_base_joint`).

### Task 2: URDF xacros (robot + mujoco_inputs + ros2_control)

**Files:**
- Create: `r2d3_mujoco/urdf/r2d3_mujoco.urdf.xacro`
- Create: `r2d3_mujoco/urdf/mujoco_inputs.urdf.xacro`
- Create: `r2d3_mujoco/urdf/ros2_control/mujoco_ros2_control.urdf.xacro`

**Interfaces:**
- Consumes: `dual_rm_description/urdf/r2d3_description.urdf.xacro` (robot body), `dual_rm_simulation/urdf/sensors/{lidar,imu,depth_camera}.urdf.xacro` (macros `lidar_sensor`, `imu_sensor`, `depth_camera_sensor` — they add links `lidar_link`, `imu_link`, `camera_optical_frame`; their `<gazebo>` tags are simply ignored by the MuJoCo converter).
- Produces: `r2d3_mujoco.urdf.xacro` accepting args `arm_model` (65b|75b, default 65b) and `headless` (default false). Its xacro output is the single `robot_description` consumed by robot_state_publisher, `ensure_mjcf.py`, and `mujoco_ros2_control` (embedded `<mujoco_inputs>` is parsed by the converter).

- [ ] **Step 1: Write `r2d3_mujoco.urdf.xacro`**

```xml
<?xml version="1.0"?>
<!--
  R2D3 Simulation URDF - MuJoCo (mujoco_ros2_control)
  Usage:
    xacro r2d3_mujoco.urdf.xacro                    # default: 65b
    xacro r2d3_mujoco.urdf.xacro arm_model:=75b     # 75b variant
-->
<robot xmlns:xacro="http://www.ros.org/wiki/xacro" name="r2d3_sim">

    <xacro:arg name="arm_model" default="65b"/>
    <xacro:arg name="headless" default="false"/>
    <xacro:property name="arm_model" value="$(arg arm_model)"/>
    <xacro:property name="arm_dof" value="${6 if arm_model == '65b' else 7}"/>

    <!-- Sensor frame macros reused from the Gazebo sim package.
         Their <gazebo> sensor tags are ignored by the MuJoCo converter;
         only the links/joints (lidar_link, imu_link, camera_optical_frame) matter. -->
    <xacro:include filename="$(find dual_rm_simulation)/urdf/sensors/lidar.urdf.xacro"/>
    <xacro:include filename="$(find dual_rm_simulation)/urdf/sensors/imu.urdf.xacro"/>
    <xacro:include filename="$(find dual_rm_simulation)/urdf/sensors/depth_camera.urdf.xacro"/>
    <xacro:include filename="$(find r2d3_mujoco)/urdf/mujoco_inputs.urdf.xacro"/>
    <xacro:include filename="$(find r2d3_mujoco)/urdf/ros2_control/mujoco_ros2_control.urdf.xacro"/>

    <!-- base_footprint (ground-level frame for Nav2) -->
    <link name="base_footprint"/>
    <joint name="base_footprint_to_base" type="fixed">
        <parent link="base_footprint"/>
        <child link="base_link_underpan"/>
        <origin xyz="0 0 0.233" rpy="0 0 ${pi/2}"/>
    </joint>

    <!-- Full robot mechanical description -->
    <xacro:include filename="$(find dual_rm_description)/urdf/r2d3_description.urdf.xacro"/>

    <!-- Sensor frames (same placement as Gazebo sim) -->
    <xacro:lidar_sensor parent_link="base_link_underpan"
                        xyz="0.0 -0.24 0.0" rpy="0 0 ${-pi/2}"/>
    <xacro:imu_sensor parent_link="base_link_underpan"
                      xyz="0 0 0" rpy="0 0 0"/>
    <xacro:depth_camera_sensor/>

    <!-- MuJoCo conversion inputs (actuators, sensors, camera, lidar, scene tweaks) -->
    <xacro:mujoco_inputs arm_dof="${arm_dof}"/>

    <!-- ros2_control with MujocoSystemInterface -->
    <xacro:mujoco_ros2_control arm_dof="${arm_dof}" headless="$(arg headless)"/>

</robot>
```

- [ ] **Step 2: Write `mujoco_inputs.urdf.xacro`**

```xml
<?xml version="1.0"?>
<!--
  MuJoCo conversion inputs for the R2D3 robot, embedded in the URDF so the
  make_mjcf_from_robot_description converter picks them up.
  raw_inputs      -> copied verbatim into the generated MJCF
  processed_inputs -> interpreted by the converter (camera, lidar, modify_element)
-->
<robot xmlns:xacro="http://www.ros.org/wiki/xacro">

    <!-- Position actuator for one arm-style joint; ctrlrange inherited from joint limits -->
    <xacro:macro name="_mj_pos_act" params="joint_name kp:=25000">
        <position name="${joint_name}" joint="${joint_name}" kp="${kp}"
                  dampratio="1.0" inheritrange="1.0"/>
    </xacro:macro>

    <!-- Gravity compensation on a body (MuJoCo analogue of the Gazebo
         <gravity>false</gravity> applied to arm/head links) -->
    <xacro:macro name="_mj_gravcomp" params="body_name">
        <modify_element type="body" name="${body_name}" gravcomp="1"/>
    </xacro:macro>

    <xacro:macro name="mujoco_inputs" params="arm_dof">
        <mujoco_inputs>

            <raw_inputs>
                <option integrator="implicitfast"/>

                <actuator>
                    <!-- Drive wheels: native velocity actuators (matches the
                         ros2_control velocity command interface) -->
                    <velocity name="joint_left_wheel" joint="joint_left_wheel"
                              kv="100" ctrlrange="-10 10"/>
                    <velocity name="joint_right_wheel" joint="joint_right_wheel"
                              kv="100" ctrlrange="-10 10"/>

                    <!-- Left arm -->
                    <xacro:_mj_pos_act joint_name="l_joint1"/>
                    <xacro:_mj_pos_act joint_name="l_joint2"/>
                    <xacro:_mj_pos_act joint_name="l_joint3"/>
                    <xacro:_mj_pos_act joint_name="l_joint4"/>
                    <xacro:_mj_pos_act joint_name="l_joint5"/>
                    <xacro:_mj_pos_act joint_name="l_joint6"/>
                    <xacro:if value="${arm_dof == 7}">
                        <xacro:_mj_pos_act joint_name="l_joint7"/>
                    </xacro:if>

                    <!-- Right arm -->
                    <xacro:_mj_pos_act joint_name="r_joint1"/>
                    <xacro:_mj_pos_act joint_name="r_joint2"/>
                    <xacro:_mj_pos_act joint_name="r_joint3"/>
                    <xacro:_mj_pos_act joint_name="r_joint4"/>
                    <xacro:_mj_pos_act joint_name="r_joint5"/>
                    <xacro:_mj_pos_act joint_name="r_joint6"/>
                    <xacro:if value="${arm_dof == 7}">
                        <xacro:_mj_pos_act joint_name="r_joint7"/>
                    </xacro:if>

                    <!-- Platform lift (prismatic, carries the torso): stiffer -->
                    <xacro:_mj_pos_act joint_name="platform_joint" kp="50000"/>

                    <!-- Head -->
                    <xacro:_mj_pos_act joint_name="head_joint1" kp="500"/>
                    <xacro:_mj_pos_act joint_name="head_joint2" kp="500"/>
                </actuator>

                <sensor>
                    <!-- IMU triple at the imu_link site (all URDF links become MJCF sites).
                         Names follow the 0.0.3 convention: <IMU>_quat/_gyro/_accel
                         for the ros2_control sensor "imu_sensor_imu". -->
                    <framequat name="imu_sensor_quat" objtype="site" objname="imu_link"/>
                    <gyro name="imu_sensor_gyro" site="imu_link"/>
                    <accelerometer name="imu_sensor_accel" site="imu_link"/>

                    <!-- Rangefinder attached to the replicated 'rf' sites generated by
                         the processed <lidar> tag below. Name must match the
                         ros2_control <sensor name="lidar"> entry. -->
                    <rangefinder name="lidar" site="rf"/>
                </sensor>
            </raw_inputs>

            <processed_inputs>
                <!-- RGB-D camera at the optical frame. fovy(deg) derived from the Gazebo
                     horizontal_fov of 1.047 rad at 640x480:
                     fovy = 2*atan(tan(1.047/2)*480/640) = 46.83 deg -->
                <camera site="camera_optical_frame" name="camera" fovy="46.83"
                        mode="fixed" resolution="640 480"/>

                <!-- 240-ray lidar matching the Gazebo gpu_lidar:
                     +/-120 deg, angle_increment = 4.1887902/239 -->
                <lidar ref_site="lidar_link" sensor_name="rf"
                       min_angle="-2.0943951" max_angle="2.0943951"
                       angle_increment="0.0175263"/>

                <!-- Gravity compensation on arm/head/torso bodies (Gazebo sim disables
                     gravity on these links). Bodies connected by fixed joints are fused
                     by the converter, so only movable-joint bodies are listed. -->
                <xacro:_mj_gravcomp body_name="l_link1"/>
                <xacro:_mj_gravcomp body_name="l_link2"/>
                <xacro:_mj_gravcomp body_name="l_link3"/>
                <xacro:_mj_gravcomp body_name="l_link4"/>
                <xacro:_mj_gravcomp body_name="l_link5"/>
                <xacro:_mj_gravcomp body_name="l_link6"/>
                <xacro:_mj_gravcomp body_name="r_link1"/>
                <xacro:_mj_gravcomp body_name="r_link2"/>
                <xacro:_mj_gravcomp body_name="r_link3"/>
                <xacro:_mj_gravcomp body_name="r_link4"/>
                <xacro:_mj_gravcomp body_name="r_link5"/>
                <xacro:_mj_gravcomp body_name="r_link6"/>
                <xacro:if value="${arm_dof == 7}">
                    <xacro:_mj_gravcomp body_name="l_link7"/>
                    <xacro:_mj_gravcomp body_name="r_link7"/>
                </xacro:if>
                <xacro:_mj_gravcomp body_name="head_link1"/>
                <xacro:_mj_gravcomp body_name="head_link2"/>
                <xacro:_mj_gravcomp body_name="platform_base_link"/>

                <!-- Wheel/caster friction tuning is added here in Task 4 after the
                     generated geom names are known (modify_element type="geom" ...) -->
            </processed_inputs>

        </mujoco_inputs>
    </xacro:macro>

</robot>
```

```python
#!/usr/bin/env python3
"""
Cached URDF->MJCF conversion for the R2D3 MuJoCo simulation.

Hashes the xacro-generated URDF plus the world file. On a cache hit the
previously generated MJCF is published directly (skipping the expensive mesh
conversion); on a miss the upstream converter is invoked, which saves the MJCF
and assets into the cache directory and publishes the model itself.

The MJCF is published on --topic with RELIABLE + TRANSIENT_LOCAL QoS, matching
the subscriber inside mujoco_ros2_control.
"""

import argparse
import hashlib
import os
import signal
import subprocess
import sys
import threading
from pathlib import Path

MJCF_FILENAME = "mujoco_description_formatted.xml"
CHECKSUM_FILENAME = "checksum"
URDF_FILENAME = "robot_input.urdf"
# Bump when converter flags in build_converter_cmd change, to invalidate caches
CONVERTER_ARGS_VERSION = "v1:save_only,add_free_joint,scene"


def compute_checksum(urdf: str, world_content: str, extra: str) -> str:
    h = hashlib.sha256()
    h.update(urdf.encode())
    h.update(world_content.encode())
    h.update(extra.encode())
    return h.hexdigest()


def cache_valid(cache_dir: Path, checksum: str) -> bool:
    mjcf = cache_dir / MJCF_FILENAME
    stored = cache_dir / CHECKSUM_FILENAME
    if not (mjcf.is_file() and stored.is_file()):
        return False
    return stored.read_text().strip() == checksum


def build_converter_cmd(urdf_file: Path, world_file: Path, cache_dir: Path, topic: str) -> list:
    return [
        "ros2", "run", "mujoco_ros2_control", "robot_description_to_mjcf.sh",
        "--save_only",
        "--add_free_joint",
        "-u", str(urdf_file),
        "--scene", str(world_file),
        "-o", str(cache_dir),
        "--publish_topic", topic,
    ]


def default_cache_root() -> Path:
    ros_home = os.environ.get("ROS_HOME", os.path.join(os.path.expanduser("~"), ".ros"))
    return Path(ros_home) / "r2d3_mujoco"


def publish_cached(mjcf_path: Path, topic: str) -> None:
    """Latch the cached MJCF on the topic and spin until shutdown."""
    import rclpy
    from rclpy.node import Node
    from rclpy.qos import QoSDurabilityPolicy, QoSProfile, QoSReliabilityPolicy
    from std_msgs.msg import String

    rclpy.init()
    node = Node("mjcf_publisher")
    qos = QoSProfile(
        depth=1,
        reliability=QoSReliabilityPolicy.RELIABLE,
        durability=QoSDurabilityPolicy.TRANSIENT_LOCAL,
    )
    pub = node.create_publisher(String, topic.lstrip("/"), qos)
    msg = String()
    msg.data = mjcf_path.read_text()
    pub.publish(msg)
    node.get_logger().info(f"Published cached MJCF from {mjcf_path} ({len(msg.data)} bytes)")
    try:
        rclpy.spin(node)
    except KeyboardInterrupt:
        pass
    finally:
        node.destroy_node()
        rclpy.try_shutdown()


def watch_and_write_checksum(topic: str, cache_dir: Path, checksum: str) -> threading.Thread:
    """Subscribe to the MJCF topic; when the converter publishes, persist the checksum."""

    def _watch():
        import rclpy
        from rclpy.node import Node
        from rclpy.qos import QoSDurabilityPolicy, QoSProfile, QoSReliabilityPolicy
        from std_msgs.msg import String

        rclpy.init()
        node = Node("mjcf_checksum_writer")
        qos = QoSProfile(
            depth=1,
            reliability=QoSReliabilityPolicy.RELIABLE,
            durability=QoSDurabilityPolicy.TRANSIENT_LOCAL,
        )
        done = threading.Event()

        def _cb(_msg):
            (cache_dir / CHECKSUM_FILENAME).write_text(checksum + "\n")
            node.get_logger().info(f"Conversion complete; checksum stored in {cache_dir}")
            done.set()

        node.create_subscription(String, topic.lstrip("/"), _cb, qos)
        while rclpy.ok() and not done.is_set():
            rclpy.spin_once(node, timeout_sec=1.0)
        node.destroy_node()
        rclpy.try_shutdown()

    t = threading.Thread(target=_watch, daemon=True)
    t.start()
    return t


def main() -> int:
    parser = argparse.ArgumentParser(description=__doc__)
    parser.add_argument("--robot-description", required=True, help="URDF XML string (xacro output)")
    parser.add_argument("--world", required=True, help="Path to the MuJoCo scene XML")
    parser.add_argument("--model", required=True, help="Robot model variant (65b|75b), names the cache dir")
    parser.add_argument("--topic", default="/mujoco_robot_description")
    parser.add_argument("--force", action="store_true", help="Recompile even if the cache is valid")
    parser.add_argument("--cache-root", default=None)
    # Tolerate ROS-injected args when run as a launch Node
    args, _unknown = parser.parse_known_args()

    cache_root = Path(args.cache_root) if args.cache_root else default_cache_root()
    cache_dir = cache_root / args.model
    cache_dir.mkdir(parents=True, exist_ok=True)

    world_path = Path(args.world)
    checksum = compute_checksum(args.robot_description, world_path.read_text(), CONVERTER_ARGS_VERSION)

    if not args.force and cache_valid(cache_dir, checksum):
        print(f"[ensure_mjcf] cache hit ({cache_dir}); skipping conversion", flush=True)
        publish_cached(cache_dir / MJCF_FILENAME, args.topic)
        return 0

    print(f"[ensure_mjcf] cache miss; converting URDF -> MJCF into {cache_dir}", flush=True)
    # Invalidate stale checksum before converting so an interrupted run stays invalid
    (cache_dir / CHECKSUM_FILENAME).unlink(missing_ok=True)
    urdf_file = cache_dir / URDF_FILENAME
    urdf_file.write_text(args.robot_description)

    watch_and_write_checksum(args.topic, cache_dir, checksum)
    child = subprocess.Popen(build_converter_cmd(urdf_file, world_path, cache_dir, args.topic))

    def _forward_sigterm(_signum, _frame):
        child.terminate()

    signal.signal(signal.SIGTERM, _forward_sigterm)
    signal.signal(signal.SIGINT, _forward_sigterm)
    return child.wait()


if __name__ == "__main__":
    sys.exit(main())
```

```python
"""
MuJoCo simulation launch for the R2D3 dual-arm mobile robot.
Mirrors dual_rm_simulation/launch/gz_sim.launch.py.
"""

import os

from launch import LaunchDescription
from launch.actions import (
    DeclareLaunchArgument,
    OpaqueFunction,
    RegisterEventHandler,
    Shutdown,
)
from launch.event_handlers import OnProcessExit
from launch.substitutions import Command, FindExecutable, LaunchConfiguration
from launch_ros.actions import ComposableNodeContainer, Node
from launch_ros.descriptions import ComposableNode
from launch_ros.parameter_descriptions import ParameterFile, ParameterValue
from ament_index_python.packages import get_package_share_directory


def launch_setup(context, *args, **kwargs):
    pkg_mujoco = get_package_share_directory("r2d3_mujoco")

    robot_model = LaunchConfiguration("robot_model").perform(context)
    world = LaunchConfiguration("world").perform(context)
    headless = LaunchConfiguration("headless").perform(context)
    force_recompile = LaunchConfiguration("force_recompile").perform(context)

    xacro_path = os.path.join(pkg_mujoco, "urdf", "r2d3_mujoco.urdf.xacro")
    robot_description_str = Command([
        FindExecutable(name="xacro"), " ", xacro_path,
        " arm_model:=", robot_model,
        " headless:=", headless,
    ]).perform(context)
    robot_description = {
        "robot_description": ParameterValue(robot_description_str, value_type=str)
    }

    controllers_yaml = os.path.join(pkg_mujoco, "config", f"controllers_{robot_model}.yaml")

    # -- MJCF provider: cached conversion (publishes /mujoco_robot_description) --
    ensure_mjcf_args = [
        "--robot-description", robot_description_str,
        "--world", world,
        "--model", robot_model,
        "--topic", "/mujoco_robot_description",
    ]
    if force_recompile == "true":
        ensure_mjcf_args.append("--force")
    ensure_mjcf = Node(
        package="r2d3_mujoco",
        executable="ensure_mjcf.py",
        name="ensure_mjcf",
        output="both",
        emulate_tty=True,
        arguments=ensure_mjcf_args,
    )

    # -- Robot State Publisher --
    robot_state_publisher = Node(
        package="robot_state_publisher",
        executable="robot_state_publisher",
        name="robot_state_publisher",
        output="screen",
        parameters=[{"use_sim_time": True}, robot_description],
    )

    # -- MuJoCo ros2_control node (physics + controller_manager + sensors) --
    remappings = [("/imu_sensor_broadcaster/imu", "/imu")]
    if os.environ.get("ROS_DISTRO") == "humble":
        remappings.append(("~/robot_description", "/robot_description"))
    control_node = Node(
        package="mujoco_ros2_control",
        executable="ros2_control_node",
        output="both",
        emulate_tty=True,
        parameters=[{"use_sim_time": True}, ParameterFile(controllers_yaml)],
        remappings=remappings,
        on_exit=Shutdown(),
    )

    # -- Controller spawners: JSB first, everything else after it exits --
    def spawner(controller):
        return Node(
            package="controller_manager",
            executable="spawner",
            arguments=[controller, "--param-file", controllers_yaml],
            output="screen",
        )

    jsb_spawner = spawner("joint_state_broadcaster")
    evt_jsb_done = RegisterEventHandler(
        event_handler=OnProcessExit(
            target_action=jsb_spawner,
            on_exit=[
                spawner("diff_drive_controller"),
                spawner("left_arm_controller"),
                spawner("right_arm_controller"),
                spawner("platform_controller"),
                spawner("imu_sensor_broadcaster"),
            ],
        )
    )

    # -- /camera/points from depth + camera_info (composable: works on Humble too) --
    pointcloud_container = ComposableNodeContainer(
        name="camera_points_container",
        namespace="",
        package="rclcpp_components",
        executable="component_container",
        composable_node_descriptions=[
            ComposableNode(
                package="depth_image_proc",
                plugin="depth_image_proc::PointCloudXyzrgbNode",
                name="point_cloud_xyzrgb",
                parameters=[{"use_sim_time": True}],
                remappings=[
                    ("rgb/camera_info", "/camera/camera_info"),
                    ("rgb/image_rect_color", "/camera/image"),
                    ("depth_registered/image_rect", "/camera/depth_image"),
                    ("points", "/camera/points"),
                ],
            ),
        ],
        output="screen",
    )

    return [
        ensure_mjcf,
        robot_state_publisher,
        control_node,
        jsb_spawner,
        evt_jsb_done,
        pointcloud_container,
    ]


def generate_launch_description():
    pkg_mujoco = get_package_share_directory("r2d3_mujoco")
    return LaunchDescription([
        DeclareLaunchArgument(
            "robot_model", default_value="65b",
            description="Robot model variant: 65b or 75b"),
        DeclareLaunchArgument(
            "world",
            default_value=os.path.join(pkg_mujoco, "worlds", "nav_empty.xml"),
            description="Full path to the MuJoCo scene XML"),
        DeclareLaunchArgument(
            "headless", default_value="false",
            description="Run MuJoCo without the Simulate window"),
        DeclareLaunchArgument(
            "force_recompile", default_value="false",
            description="Force URDF->MJCF recompilation even if cached"),
        OpaqueFunction(function=launch_setup),
    ])
```
